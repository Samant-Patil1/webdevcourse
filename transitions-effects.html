GSAP Basics

GSAP is the industry standard for web animation, powering sites for Google, Netflix, Microsoft, and thousands more. It handles everything from simple fades to complex sequenced animations with unmatched performance.

🎯 Core Methods

  • gsap.to(target, vars) — Animate TO a state from current values
  • gsap.from(target, vars) — Animate FROM a state to current values
  • gsap.fromTo(target, from, to) — Full control with explicit from/to states
  • gsap.timeline() — Chain multiple animations in sequence
  • gsap.set(target, vars) — Instantly set properties without animation

⚡ Essential Properties

  • duration — Animation length in seconds
  • delay — Wait before starting
  • ease — Timing function (power2.out, elastic, bounce)
  • stagger — Delay between multiple targets
  • repeat — Number of repeats (-1 for infinite)
  • yoyo — Alternate direction on repeat
  • onComplete — Callback when done
💡 Pro Tip: Use defaults: { ease: "power2.out" } in timelines to set default easing for all child animations. Always prefer GSAP easing over CSS for better control and performance.

Page Transitions & Load Animations

First impressions matter. Smooth page load sequences and transitions create a polished, professional feel that keeps users engaged from the moment they land on your site.

🚀 Common Patterns

  • Loader with Progress — Show loading progress before revealing content
  • Staggered Text Reveal — Animate characters or words one by one
  • Curtain/Fade Transitions — Smooth overlays between pages
  • Content Stagger — Hero → Subtitle → CTA in sequence
  • Ripple Effects — Material Design-inspired button feedback

📋 Implementation Tips

  • Use transform-origin for scale animations that feel natural
  • Set initial states with gsap.set() to prevent flash of unstyled content
  • Use onComplete callbacks for navigation after animations
  • Consider reduced motion preferences with prefers-reduced-motion
💡 Pro Tip: Combine multiple small animations in a timeline rather than one big animation. It feels more sophisticated and gives you finer control over timing.

ScrollTrigger & Scroll Animations

ScrollTrigger is GSAP's plugin for scroll-driven animations. It enables you to trigger animations based on scroll position, pin elements, create parallax effects, and build immersive storytelling experiences.

🎯 Key Concepts

  • trigger — The element that triggers the animation
  • start/end — When to begin/end (e.g., "top 80%", "center center")
  • scrub — Link animation progress to scroll position
  • pin — Fix element in place during animation
  • toggleActions — Control play/pause on enter/leave

📊 Start/End Position Syntax

"top bottom"    // Top of trigger hits bottom of viewport
"center center" // Centers align
"top 80%"       // Top of trigger at 80% down viewport
"+=500"         // 500px after start

🎮 Toggle Actions

Format: "onEnter onLeave onEnterBack onLeaveBack"

  • play — Start animation
  • pause — Stop animation
  • resume — Continue from current position
  • reverse — Play backwards
  • restart — Go to beginning and play
💡 Performance Tip: Always use transforms (x, y, scale, rotation) instead of layout properties (top, left, width) for smoother 60fps animations. Add will-change: transform to animated elements.
