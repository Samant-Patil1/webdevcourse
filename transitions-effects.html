GSAP Basics

GSAP is the industry standard for web animation, used by Google, Netflix, and Microsoft. It handles everything from simple fades to complex sequenced animations.

Core Methods

  • gsap.to() - Animate TO a state
  • gsap.from() - Animate FROM a state
  • gsap.fromTo() - Full control with from/to
  • gsap.timeline() - Chain animations
💡 Pro Tip: Use GSAP's built-in easing functions for better performance and control than CSS.
