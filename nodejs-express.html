Node.js Introduction

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows you to run JavaScript on the server side.

Why Node.js?

  • JavaScript Everywhere - Same language for frontend and backend
  • Non-blocking I/O - Handles many connections efficiently
  • NPM Ecosystem - Largest package registry
  • Fast Execution - V8 engine compiles to machine code

Core Modules

  • http/https - Create web servers
  • fs - File system operations
  • path - File path utilities
  • os - Operating system info
💡 Pro Tip: Use nodemon during development to auto-restart your server when files change.
